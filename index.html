Cosmic is a single runtime for broad applications.

Desktop and Web UI
  • Declarative widget framework.
  • Builtin widgets library.
  • Styles and themes.
  • Runs on desktop with native graphics/input.
  • Runs on the web with Canvas/WebGL.
  • Soon
Games and real-time apps
  • 2D Graphics API. *
  • 3D Graphics API. *
  • Video/Sound API.
  • Input API.
  • Soon

* Supports OpenGL, and soon WebGPU.

Backend development
  • Web server API.
  • HTTP client API.
  • Network API.
  • File system API.
  • Soon
Scripting and tooling
  • Go fast for all your scripting needs.
  • Build command line tools.
  • Automate workflows and tasks.
  • Soon

Cosmic is a single tool to install and learn.

Developer tools
  • Secure runtime with broad or granular permissions.
  • Builtin test runner.
  • Builtin hot code reloading.
  • Builtin profiling.
  • Import external code using ES6 modules.
  • Debug code with Chrome DevTools.
  • Soon
Curated standard library
  • Simple JS API* with sync/async ops.
  • Will be backwards compatible starting at (v1.0).
  • Online and offline docs.
  • Typed definitions for your text editor.
  • Soon

* The same API will also be available to WASM/WASI.

TypeScript support
  • Opt-in to types for projects that scale.
  • Builtin TypeScript compiler focused on speed.
  • Soon
Distribution tools
  • Bundle your software into a single binary for distribution.
  • Single code base for multiple platforms.
  • Protect your users with permission controls.
  • Soon

Modern technologies
  • Embrace quality software tools and patterns.

  • Marketable skillset for your career.
Reliable engine
  • Cosmic is open source and released under the MIT License.
